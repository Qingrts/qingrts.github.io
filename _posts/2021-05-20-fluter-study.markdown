# 查看当前所有终端

```cmd
flutter devices
```

# 给当前项目添加web支持
```cmd
flutter create .
```

# 在chrome上测试应用
```cmd
flutter run -d chrome
```

# 打包apk文件
```cmd
flutter build apk
```

会在项目目录中的build/app/outputs/apk/release下生成对应的文件
